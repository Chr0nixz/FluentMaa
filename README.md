## 关于此GUI
你觉得Maa原版的GUI十分的朴素？  
你是否因为Maa不能多开而烦恼？  
你是否因为Wpf无法跨平台而忧心？  
快来使用FluentMaa！
- 现代的界面
- 简易的多开
- 方便的跨平台（虽然还没有进行测试）
- 更新无需重启
- 更小的内存占用
- 剩下的以后再说
